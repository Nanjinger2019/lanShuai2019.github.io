2019-09-03-JavaScript学习笔记(一)

发表于 2019-09-04 | 分类于 前端 ， JavaScript ， 查漏补缺 | | 阅读次数:

虽然从大学时代到现在，做了不少的Web项目，但基本上是用什么学什么，从来没有系统性地学习过javaScript、Vue等等前端知识，这一点的危害从校招的溃败中体现的淋漓尽致。痛定思痛，我将通过博文来记录我的JavaScript学习历程。

在HTML中使用JavaScript

DOM、BOM

DOM

DOM是文档对象模型，是用于HTML的应用程序编程接口。DOM把整个页面映射为一个多层节点结构，HTML或者XML页面中每个组成部分都是某种类型的节点。
通过DOM创建的树形结构，使开发人员获得了控制页面内容和结构的权力。借助DOM提供的API，开发者可以方便地CRUD各种页面组成部分。

BOM

BOM是浏览器对象模型，BOM用于访问和操作浏览器显示的页面以外的部分。人们习惯上把所有针对浏览器的JavaScript扩展算是BOM的一部分。比如

  • 提供浏览器详细信息的navigator对象
  • 通过浏览器所加载页面详细信息的localtion
  • 提供用显示器分辨率详细信息的screen对象
  • 对cookie的支持

script元素属性

script元素有六个属性

  • async: 异步下载脚本，不妨碍页面中其他操作，支队外部脚本有效
  • charset: 表示通过src属性指定代码的字符集
  • defer: 表示脚本可以延迟到文档完全被解析和显示之后再执行。
  • src: 要执行代码的外部文件
  • type: 编写代码使用的脚本语言和内容类型

defer 与async的不同：
同时被defer修饰的标签，将按照前后顺序执行，而被async修饰的标签，不保证其之间的执行顺序。

文档模式

IE5.5引入文档模式的概念，这个概念是通过使用文档类型doctype实现的，最初的两种文档模式是混杂模式和标准模式,混杂模式会让IE的行为与IE5相同，而标准模式
则让IE的行为更接近标准行为。虽然这两种模式主要影响CSS内容呈现，但是也会影响到JS的执行。

基本概念

语法

JavaScript区分大小写，定义标识符时第一个字符必须是

  • 字母
  • 下划线(_)
  • 美元符号($)

其他字符还可以是数字。
JavaScript标识符按照惯例采用驼峰大小写格式。
ES5引入严格模式：一些不确定的行为将得到处理，某些不安全的操作也会跑出错误。整个脚本启用严格模式，可以在顶端添加一个编译指示

在函数内部上方包含这条编译指示，可以指定函数在严格模式下执行：

数据类型

五种简单类型：Undefined、Null、Boolean、Number、String
一种复杂类型：Object

Undefined类型

变量声明未初始化

Null类型

null表示一个空对象指针，

如果定义的变量准被在将来用于保存对象，最好初始化为null而不是其他值，这样，只要检查null值就能知道对应的变量是否一斤保存了一个对象的引用。
undefined派生自null，所以

console.log(null == undefined); // true
console.log(null === undefined); // false

Boolean类型

只有true和false，和1与0值相等，但类型不同。可以使用函数Boolean(avg)强制转化为Boolen类型。

Number类型

使用IEEE754格式来表示整数和浮点数。为了支持各种数据类型，提供字面量格式。

var intNum = 55 // 整数
var octalNum = 070 // 八进制56 ，严格模式下无效
var hexNum = 0xA; // 十六进制10
var floatNum = 1.1 // 浮点数，必须包含一个小数点
var floatNum2 = 1.0 // 整数，解析为1，因为保存浮点数内存空间是证书2倍，ES会不失时机地将浮点数转化为整数
var floatNum3 = 3.125e7 // 等于31250000

NaN是一个特殊的数值，表示一个本来要返回数值的操作数未返回数值的情况。

NaN == NaN // false
isNaN(NaN) // true
isNaN(10) // false, 10是一个数值
isNaN(“10”) // false, 可以被转化为一个) // true, 不能被转化为一个数值
isNaN(true) // false, 可以被转化为1

String类型

ES中单双引号形式均可。
ES中字符串不可变，一旦创建，值不能变。如果要改变，首先要销毁原字符串，然后再用另一个包含新值的字符串填充该变量。
使用toString()方法将变量转化为字符串。
String(avg)方法：

  • 如果值有toString()方法，调用该方法
  • 如果值为null，返回”null”
  • 如果值为undefined，返回”undefined”

语句

for-in 语句

for-in语句是一种精准的迭代语句，可以用来枚举对象的属性

for (property in expression) statement

例如，循环显示window对象的所有属性。

ES中的属性没有顺序，因此，通过for-in循环输出的属性名的顺序是不可预测的。所有属性都会被返回一次，但返回的前后次序可能会因浏览器而异。

变量、作用域、内存

基本类型和引用类型的值

JavaScript只能给应用类型值动态添加属性，不能给基本类型值添加属性。
当从一个变量向另一个变量复制基本类型的值，会在变量对象上创建一个新值，然后把该值复制到为新变量分配的位置上，分配在栈中；
当从一个变量向另一个变量复制引用类型的值，只是将新值指向原来值相同的存储在堆中的一个对象。
ES中所有函数的参数都是按值传递，如果参数是什么类型，传参均如复制，遵循上面两个规则。

栈和堆区别
栈由系统分配，堆由程序员分配；
栈向低地址扩展，是一块连续的内存区域，堆向高地址扩展，使用链表作为数据结构，是不连续的内存区域；
栈由系统分配，速度快；堆由程序员自己分配，速度慢，容易产生碎片；

虽然在检测基本类型时，typeof比较有用，但是在检测引用类型是，需要用instanceof操作符，因为我们并不是想知道某个值是对象，而是想知道他是什么类型的对象。

alert(person instanceof Object) // 变量person是Object吗？
alert(person instanceof Array) // 变量person是Array吗？
alert(person instanceof RegExp) // 变量person是RegExp吗？

根据规定，所有引用类型的值都是Object的实例。

内部环境可以通过作用域链访问所有外部环境，外部环境不能访问内部环境的任何变量和函数。每个环境都可以向上搜索作用域链，以查询变量和函数名；但任何环境都不能通过向下搜索作用域链而进入另一个执行环境。

垃圾收集

标记清除

变量进入环境标记进入，离开环境标记离开。垃圾收集器运行时会给存储在内存中所有变量都加上标记，然后，去掉环境中的变量以及环境中变量引用的变量的标记。在此之后再被加上标记的变量将被视为准备删除的变量，因为环境中的变量已经无法访问到这些变量了。

引用计数

跟踪记录每个值被引用的次数，当声明一个变量并将一个引用类型值赋给该变量时，引用次数+1，如果该变量又被赋予另一个值，则这个值的引用次数-1，当这个值的引用次数减为0时，说明没有变量能找到这个值，将其空间回收。

引用类型

object类型

访问对象属性通常使用点表示法，JavaScript中也可以使用方括号表示法来访问对象的属性。在使用方括号表示法时，应该将要访问的属性以字符串的形式放在方括号中。方括号表示法的有点是可以通过变量来访问属性，变量中可以包含空格。

Array属性

ES数组每一项可以保存任何类型的数据。
数组的length属性不是可读的，通过设置这个属性，可以从数组末尾移除项或者项数组中添加新项。如果将length属性设置为大于数组项数的值，则新增的每一项都会取得undefine值。

let colors = new Array(‘green’, ‘blue’)
colors[colors.length] = ‘blue’ // 向数组尾部添加新项

检测数组

转换方法

  • valueOf(): 返回数组本身
  • toString(): 返回有数组中各个值的字符串形式拼接而成的一个一逗号分隔的字符串。实际上，为了创建这个字符串会调用数组中每一项的toString()方法
  • toLocaleString(): 和valurOf()以及toString()的唯一区别是调用每个数组项的toLocaleString()方法。

    join(avg) : 只接受一个参数，即用作分隔符的字符串，然后返回包含所有数组项的字符串。
    如果不给join()方法传入任何值，或者给它传入undefined，则使用逗号作为分隔符。

栈方法

  • push(); 可以接受任意数量的参数，把它们逐个添加到数组末尾，返回修改后数组的长度。
  • pop(): 从数组末尾移除最后一项，较少数组的length值，然后返回被移除的项。

队列方法

  • push(): 同上
  • shift(): 从数组中移除第一项，返回被移除的数组项。

    unshift(): unshift(): 在数组前端添加若干个项并返回新数组长度。

重排序方法

  • reverse(): 反转数组项的顺序，返回反转后的数组。
  • sort(): 调用数组每一项的toString()方法，然后比较得到的字符串，已确定如何排序，即使数组中每一项都是数值，sort()方法比较的也是字符串。
    sort()函数可以接收一个比较函数作为参数，便于我们指定哪个值位于哪个值的前面。
    如果第一个参数应该位于第二个参数之前则返回一个负数，反之，返回正数，相等，返回0。
    这个比较函数可以适用于大多数数据类型，只要将其作为参数传递给sort方法即可。

    对于数值类型或者valueOf()方法会返回数值类型的对象类型，可以使用一个更简单的比较函数

操作方法

  • concat(): 基于当前数组中的所有相创建一个新数组，先创建数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。在没有给concnat()方法传递参数的情况下，他只是复制当前数组并返回副本。
    如果传递给concat()方法的是一或多个数组，则该方法会将这些数组中的每一项添加到结果数组中。如果传递的不是数组，这些值会被简单地填在到结果数组的末尾。
  • slice(start,end): 只有一个参数，返回start到之后，有两个参数，返回[start, end)
  • splice(start, length, replaceItems…): 从start开始，之后的length个数组项，替换为repalceItmes(可以是多个项，会被当做一个数组来处理)。splice()方法返回一个数组，该数组包含从原始数组中删除的项，如果没有删除任何项，则返回一个空数组。

位置方法

  • indexOf(): 接收两个参数，要查询的项、查找起点位置的索引， 从数组的开头开始向后查找。
  • lastIndexOf(): 参数同上，两个参数的时候，从起始位置向前查询。

迭代方法

都会接收三个参数，item(数组项)、index(该项在数组中的位置)、array(数组本身)，对数组中每一项运行给定函数。

  • every(): 如果该函数对每一项都返回true， 则返回true；
  • filter(): 返回会返回true的项组成的数组；
  • forEach(): 没有返回值；
  • map(): 返回每次调用结果组成的数组；
  • some(): 任一数据项返回true，则返回true；

归并方法

  • reduce()和reduceRight():迭代数组的所有项，然后构建一个最终返回的值，都接收两个参数：一个在每一项上调用的函数和（可选的）作为归并基础的初始值。
    reduce()从第一项开始，reduceRight()从最后一项开始。都接收4个参数，prev(前一个值)、cur(当前值)、index(项的索引)、array(数组对象)，这个函数返回的任何值都会作为第一个参数自动传给下一项。

Date 类型

  • Date.now(): 返回当前时间自1970年1月1日凌晨经过的毫秒数，可以用于分析代码运行时间，简化了使用Date对象分析代码的工作。
  • getFullYear(); 返回四位数年
  • getMonth(): 返回月，0-11
  • getDate(): 返回日，1-31
  • getHours(): 返回小时，0-23
  • getMinutes(): 返回分钟，0-59
  • getSeconds(): 返回秒，0-59

RegExp类型

  • g: 全局模式
  • i: 不区分大小写
  • m: 多行模式

    RegExp实例方法

  • exec(): 捕获组。接受一个参数，即要应用模式的字符串，返回包含第一个匹配项信息的数组，没有匹配项返回null，此数组区别于其他Array实例，有两个额外的属性：index(匹配项在字符串中的位置)和input(应用正则表达式的字符串)

    pattern.exec(text);

  • test(): 接收一个字符串参数，匹配返回true，反之返回false。

Function类型

函数声明和函数表达式

解析器会率先读取函数声明，使其能够在执行任何代码前可用；但是函数表达式必须等到解析器执行到它所在的代码行，才会真正被解释执行。

函数内部属性

arguments

arguments是函数内部的一个特殊的对象，这个对象有一个名叫callee，该属性是一个指针，指向拥有这个arguments对象的函数。
在递归函数中，我们可以使用arguments.callee()来替代原函数，这样，即使原函数名称指向别处或者释放，函数也不用改。

this

this是函数内部另一个特殊的对象，this引用的是函数执行的环境对象，也就是当前函数所在的作用域。当在网页的全局作用域中调用函数时，this对象引用的就是window

函数属性和方法

每个函数都包含两个属性：length(函数希望接收的命名参数个数)、prototype.
对于ES中的引用类型而言，prototype是保存他们所有实例方法的真正所在。诸如toString()、 valueOf()等方法实际上都保存在prototype名下，只不顾是通过各自对象的实例访问罢了。
prototype不可枚举，使用for-in无法发现。
每个函数都包含两个非继承而来的方法：apply()和call()。这两个方法的用途都是在特定的作用域中调用函数，实际上等于设置函数体内this对象的值。

  • apply(): 接收连个参数：一个是在其中运行函数的作用域， 另一个是参数数组(可以是Array实例，也可以是arguments对象);
  • call(): 作用同上，区别在于call()方法的参数必须逐个列举出来。

    fun.apply(environment, arguments): 在environment中寻找fun函数，将arguments作为参数给其运行。

  • bind(): 创建一个函数实例，其this值会被绑定到传给bind()函数的值。

基本包装类型

可用于格式化数值的方法是toExponential(); 返回以指数表示法表示的数值的字符串形式，与toFixed()一样，toExponential也接收一个参数，也是输出结果中的小数位数。

String类型

两个用于访问字符串中特定字符的方法：

  • charAt(); 接收一个参数，即索引。
  • charCodeAt(); 同上，不过返回的不是字符而是字符编码。

ES提供三个基于字符串创建新字符串的方法,都返回一个新字符串：

  • slice(start, end); 返回[start, end)
  • substring(start, end): 返回[start, end)
  • substr(start, length): 返回[start, start + length)

字符串位置方法有两种，找到第一个之后返回索引；并且都可以接受第二个参数：开始查找的位置。每次找到pos后从pos + 1开始继续找，知道pos变成-1，这样就可以把所有匹配的项都找出来。

  • indexOf(): 从头找
  • lastIndexOf(): 从尾找

删除前后空格：

  • trim(): 返回字符串删除前后空格之后的副本。

字符串大小写转化：

  • toLowerCase(): 转小写
  • toUpperCase(): 转大写

字符串模式匹配方法：

  • match(): 接收一个参数，要不是一个正则表达式，要不是一个RegExp对象。返回与正则表达式中的捕获组匹配的字符串。
  • search(): 参数也是一个正则表达式或者一个RegExp对象，返回第一个匹配项的索引，没有则返回-1；
  • replace(): 两个参数：第一个是一个RegExp对象或者一个字符串，第二个参数可以是一个字符串或者一个函数；

    replace方法的第二个参数也可以是一个函数，在只有一个匹配项的情况下，会向这个函数传递三个参数：模式的匹配项、模式匹配项在字符串中的位置和原始字符串。
  • split(): 基于指定的分隔符将一个字符串分割成多个子字符串，并将结果放在一个数组中。

比较字符串：

  • localeCompare(): 参数为一个字符串，如果该字符串的字典序在调用localeCompare方法的字符串之前，则返回一个正数；相等，返回0；在其之后，则返回一个负数；

Global对象

不属于任何其他对象的属性和方法，最终都是它的属性和方法。诸如isNaN()、isFinite()、parseInt()、parseFloat()都是Global对象的方法。

eval方法

eval()方法就像一个完整的ECMAScript解析器，只接受一个参数，即要执行的ECMAScript字符串。在eval()中创建的任何变量或者函数都不会被提升，因为在解析代码的时候，它们被包含在一个字符串中；它们只在eval()执行的时候创建。
严格模式下，在外部访问不到eval()中创建的任何变量或函数。

Math对象
